```
百度 20k-40k 上海
1.精通HTML/XHTML,CSS,熟悉页面架构和布局,对Web标准和标签语义化有深入理解;
2.精通Ajax,JavaScript,DOM等前端技术,掌握面向对象编程思想;
3.对Web技术创新及丰富互联网应用开发有浓厚兴趣,了解单页面应用(SPA);
4.对用户体验,交互操作流程,及用户需求有深入理解;
5.极强的团队精神,优秀的学习能力与创新能力;
6.了解或精通node.js,angularjs,react,redux,es6,jQuery,requirejs等更佳;
7.有扎实的计算机原理,算法,网络,数据库,操作系统等基础,有扎实的代码功底;
8.有良好的沟通与表达能力,学习能力,动手能力与逻辑分析能力;
9.有React/Angular/Vue等前端框架的使用经验者优先,有小程序开发经验者优先,有Web后端设计开发经验者优先;
10.熟悉HTML5,CSS3,能够基于移动终端浏览器实现基本的交互效果;
11.熟练使用原生JavaScript;
```
